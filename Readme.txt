使用方法：
0，使用命令行进入此目录下，或在此目录下打开命令行
1，输入make，点击回车
2，输入./creatNum 20000，点击回车
3，输入./testBubble，点击回车

说明：
creatNum用于生成0~19999共20000个从小到大排列的自然数
testBubble用于对以上的20000个自然数先按从大到小的顺序排列，再将排好的数字按照从小到大的顺序排列，排序使用冒泡法

完整使用方法：
...
2，输入./creatNum int，这里的int替换为大于0小于INT_MAX的整数
3，输入./testBubble filename，这里的filename替换为包含要排序的整数数据的文件名称
